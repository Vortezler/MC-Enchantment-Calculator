CC0 item icons
==============

These pixel-art item icons are from the "496 pixel art icons for
medieval/fantasy RPG" set on OpenGameArt, released under CC0 1.0
(Public Domain Dedication) by the artist.

  Source:  https://opengameart.org/content/496-pixel-art-icons-for-medievalfantasy-rpg
  License: CC0 1.0 Universal (public domain) — no attribution required.

Files were selected from that set and renamed to match the calculator's
item ids (sword.png, axe.png, ...). Because the pack is generic fantasy
art, items with no close match (pickaxe, shovel, hoe, crossbow, fishing
rod, elytra, shield) intentionally fall back to the app's own generated
pixel-art icons.

These are original CC0 art, NOT Minecraft's textures — safe to redistribute.
