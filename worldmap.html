How this works This map is generated right here in your browser from your seed — nothing leaves your device. It runs cubiomes, an open-source (MIT) reimplementation of Minecraft Java's world generation, compiled to WebAssembly. Its biome output is verified against cubiomes' own Minecraft-validated test vectors. Drag to pan, scroll to zoom, hover to read the biome and coordinates, and use Find the nearest… to jump to the closest structure from your position.
World generation by cubiomes (MIT), running locally via WebAssembly — see CUBIOMES-LICENSE.txt. The native engine covers Java Overworld; Bedrock, and the Nether/End, fall back to the Chunkbase cross-check link. A few 1.18+ structures (desert/jungle temples, mansions) can rarely differ from in-game because the library can't run surface-height checks. Not affiliated with Chunkbase or Mojang.
Text seeds are hashed exactly like Minecraft does, so they work too. To double-check any result, use the Chunkbase cross-check link above.
